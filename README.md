### Workflow ###

Once:
- Load /extension/chrome as unpacked extension into Chrome.
- npm run start

After Every change:
* npm run bundle
* (If you made changes to HOST_DOMAIN or es6_run_exy.js):
  * npm run bundle_chrome
  * Reload extension at chrome://extensions
* Reload web page
